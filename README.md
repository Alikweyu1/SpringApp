# User Details
## Get All Accounts:
1. Account Owner should be present for all the users (Normal, SSO/Generic).
   a. For SSO/Generic users: The Bank ID would be mentioned in two fields i.e., Account Owner and Bank ID from the UI. The value mentioned against the Account Owner field through UI should populate against the Account Owner tag in API Response.
   b. For Normal users: Bank ID added should be populated against Account Owner in the API Response.
